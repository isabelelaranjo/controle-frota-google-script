# controle-frota-google-script
Sistema de agendamento de veículos com validação de rodízio integrado ao Google Sheets.

Projeto desenvolvido para gerenciar o uso de veículos da empresa, com foco em facilitar o agendamento e evitar multas de rodízio.

## ✨ Funcionalidades
- **Agendamento Online:** Interface responsiva para mobile (iPhone/Android).
- **Validação de Rodízio:** Bloqueio automático de reservas nos horários de pico (Seg/Qua).
- **Calendário Integrado:** Visualização mensal das reservas (FullCalendar API).
- **Banco de Dados:** Integração total com Google Sheets.

## 🛠️ Tecnologias
- Google Apps Script (JavaScript/HTML/CSS)
- FullCalendar API
- Google Sheets API
